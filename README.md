# 📚 Projetos e Exercícios

Este repositório reúne meus projetos e exercícios desenvolvidos em **HTML, CSS e JavaScript**, com foco em aprendizado e prática de desenvolvimento web.

---

## 🚀 Projetos

### 🔄 Conversor de Moedas
- **Descrição:** Aplicação simples para converter valores entre **Real (BRL)**, **Dólar (USD)** e **Euro (EUR)**.
- **Tecnologias:** HTML, CSS, JavaScript
- **Funcionalidades:**
  - Conversão entre moedas com taxas fixas
  - Botão para inverter moedas
  - Interface estilizada e responsiva
- **Próximos passos:**
  - Integrar API de câmbio para valores reais
  - Melhorar responsividade em dispositivos móveis

### 🌐 Landing Page - Outlier Open Space
